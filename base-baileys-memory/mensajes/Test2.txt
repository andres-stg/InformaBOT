📒 _*MANUAL* DE TÉRMINOS PARA DIFERENCIAR LA DESINFORMACIÓN._ 


*1- Desinformación:* Contenido falso o engañoso que se difunde deliberadamente para confundir o manipular.
*2- Información errónea:* Información incorrecta o inexacta que se comparte sin intención de causar daño.
*3- Información maliciosa:* Tomar información real para manipularla con el objetivo de causar daño o perjuicio.
